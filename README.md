Developed Cravly, a responsive food delivery website using HTML, CSS, and JavaScript.
Implemented a dynamic cart system with add/remove items, quantity control, and real-time total calculation.
Built real-time search, filtering, and sorting features to enhance user experience.
Utilized localStorage for persistent cart data across sessions.
Designed a modern, brown-themed UI with responsive layout and smooth user interactions.
Simulated an order placement system with success feedback and cart reset functionality.

Features :-
Responsive design (works on mobile & desktop).
Food items displayed in card layout (image, price, rating).
Real-time search functionality.
Filter (veg/non-veg, price range) and sorting (price, rating).
Dynamic Add to Cart system (add/remove, quantity control).
Automatic total price calculation.
Cart data persistence using localStorage.
Order placement simulation with success message.
Smooth UI interactions (hover effects, transitions).

Advantages :-
Provides a realistic user experience similar to modern food delivery platforms.
Demonstrates strong JavaScript fundamentals.
Shows ability to manage client-side state without frameworks.
Clean and responsive UI improves usability and accessibility.
Lightweight project , easy to run and test.
